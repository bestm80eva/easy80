Mario's Small Z80 system

Here is a small Z80 system. This project is good for first time, Z80 hardware and software home study.
The original project and page is made by Mario Prato from Italy.

Minimum Z80 system with:

  • Clock 4 MHz
  • 8 K EPROM
  • 32 K RAM
  • 8 bit output port
  • 8 bit input port
